FAQ

What exactly am I buying?

You are buying a usage license for a CodeBell software system, including hosted deployment and access during the validity period.

Is support included in the license fee?

Support is offered per product sale and is quoted separately based on SLA, scope, and add-ons.

Do you give the source code?

No. The license provides software usage rights and deployment. Source code is not sold or transferred.

Where is the system hosted?

The system is hosted under CodeBell deployment by default. Private hosting and special environments can be quoted.

Can we customize features or add integrations?

Yes. Custom modules, reports, workflows, WhatsApp/SMS, POS, and mobile apps can be added as paid enhancements.
